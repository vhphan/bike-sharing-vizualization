# bike-sharing-viz
Leaflet vue dashboard using bike sharing data. Data from
https://www.kaggle.com/jboysen/austin-bike


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
